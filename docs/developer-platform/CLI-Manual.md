# CLI Manual

## AEOS commands

- `aeos doctor` validate local environment and required env vars.
- `aeos init` initialize repository defaults.
- `aeos new service <name>` scaffold FastAPI service template.
- `aeos new package <name>` scaffold Python package template.
- `aeos new app <name>` scaffold React app template.
- `aeos new agent <name>` scaffold AI agent template.
- `aeos generate crud` structured guidance for CRUD generation flow.
- `aeos generate api` structured guidance for API generation flow.
- `aeos migrate` run migration target.
- `aeos seed` structured guidance for seed flow.
- `aeos docs` generate OpenAPI files into `docs/openapi`.
- `aeos lint` run lint target.
- `aeos test` run test target.
- `aeos build` run build/dev target.
- `aeos release` structured guidance for release process.
