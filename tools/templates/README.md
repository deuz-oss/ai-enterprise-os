# Templates

Reusable generator templates used by AEOS CLI.

- `fastapi-service/` canonical service template with health/version/metrics endpoints.
- `react-app/` frontend application template.
- `flutter-app/` mobile application template.
- `python-package/` reusable Python package template.
- `typescript-package/` reusable TypeScript package template.
- `ai-agent/` AI agent module template.
