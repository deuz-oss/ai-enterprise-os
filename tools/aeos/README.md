# AEOS CLI

AEOS CLI is the standard developer command interface for AI Enterprise OS.

## Run

From repository root:

```bash
python tools/aeos/src/aeos/cli.py --help
python tools/aeos/src/aeos/cli.py doctor
```

## Commands

- `doctor`
- `init`
- `new service|package|app|agent`
- `generate crud|api`
- `migrate`
- `seed`
- `docs`
- `lint`
- `test`
- `build`
- `release`
